Why Artemis

Everything you need,
nothing you don't.

Built from scratch for Roblox with a focus on aesthetics, performance, and developer experience.

Frosted Glass Design
Deep dark palette with translucent surfaces, soft accent glows and layered shadows that look premium in any game.
Smooth Animations
Every interaction uses Quint and Back easing curves. Toggles, sliders, dropdowns and tab switches all feel alive.
Fully Themeable
Pass any Accent color to Window() and every glow, border and pill updates automatically.
Zero Dependencies
One Module, no external packages. Drop it in ReplicatedStorage and require it from any client script.
Notifications
Slide-in toast notifications with a countdown progress bar, configurable duration and left/right side positioning.
Toggle Keybind
Assign any key to show and hide the window. Uses a clean Visible toggle so all state is preserved.

One line to get started.

Refer ArtemisUI as a loadstring, then call it as a module.

Script.lua 
local A = loadstring(game:HttpGet'https://artemis.valencea.xyz/loader.lua')()

local win = A:Window({
    Title  = "MyPanel",
    Accent = Color3.fromRGB(108, 99, 255),
})

win:SetToggleKey(Enum.KeyCode.RightShift)
win:Notify("Loaded", "RightShift to toggle.", 4, "right")

local tab = win:Tab("Main")
local sec = tab:Section("Combat")

sec:Button("Kill Aura", function()
    print("activated")
end)

local toggle = sec:Toggle("ESP", false, function(v)
    print("ESP:", v)
end)

local spd = sec:Slider(
    "Walk Speed", 16, 300, 16,
    function(v)
        local hum = game.Players.LocalPlayer
            .Character:FindFirstChild("Humanoid")
        if hum then hum.WalkSpeed = v end
    end
)

sec:Dropdown("Team",
    {"Red", "Blue", "Neutral"},
    function(v) print(v) end
)

-- reset everything programmatically
local reset = tab:Section("Actions")
reset:Button("Reset All", function()
    toggle:Set(false)
    spd:Set(16)
    win:Notify("Reset", "Defaults restored.", 3, "right")
end)
